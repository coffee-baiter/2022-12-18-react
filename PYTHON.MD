# PYTHON

## variables
<code>
  x = 1
  x = x + 1
  print(x) ### 3
</code>

## Strings
<code>
  x = 'abc'
  y = 'de'
  z = x + y
  z ### 'abcde'
  z = z[1:3]
  z ### 'bc'
</code>

## Functions
<code>
  def f(x):
    y = x + 2
    return y
  f(5) ### 7
</code>

## IF Statements
<code>
  x = 2
  if x + 1 == 3:
    print(x) ### 2
</code>

## Loops
<code>
  for i in range(1, 100):
    if i % 2 == 0:
      print(i) ### 2 4 6 8 10 12 14 ...... 94 96 98
</code>
